# Dashboard Mundial 2026 - Hora Ecuador

Paquete listo para publicar en GitHub Pages.

## Estructura

- `index.html`: dashboard principal.
- `manifest.json`: configuración tipo app/PWA.
- `icons/icon-192.png`: ícono para iPhone y navegador.
- `icons/icon-512.png`: ícono principal PWA.

## Publicar en GitHub Pages

1. Crea un repositorio en GitHub, por ejemplo: `mundial-dashboard`.
2. Sube estos archivos respetando la estructura de carpetas.
3. Entra a `Settings` → `Pages`.
4. En `Build and deployment`, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guarda y espera a que GitHub genere el enlace.

## Agregar al inicio del iPhone

1. Abre el enlace de GitHub Pages en Safari.
2. Toca `Compartir`.
3. Selecciona `Agregar a pantalla de inicio`.
4. El acceso aparecerá como `Mundial EC` con ícono personalizado.
